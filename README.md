# Blackjack Simulator 
## Project Overview
This project is a text-based Blackjack simulator written in Python3. 
The goal is to create a simplified version of Blackjack, where the player competes against the dealer.
The game focuses on core mechanics, including dealing cards, calculating hand values,  and determining the winner.

## Features
- 6 52-card decks shuffled together.
- Card dealing to the player and dealer.
- Hand values with handling for Aces(1 or 11)
- Player actions:
    - Hit
    - Stand
- Playing multiple rounds.
- Basic statistics:
    - Number of games played
    - Win/Loss ratio
